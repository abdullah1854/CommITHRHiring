## Product Goal
This is an internal HR recruitment system used by:
- Admin
- HR / Recruiter team

The platform should help a small HR team manage hiring more efficiently using AI. The AI should support first-level screening by analyzing job descriptions and resumes, scoring candidates, ranking them, suggesting interview questions, summarizing candidate profiles, and highlighting skill gaps.

Important:
- AI should recommend by default, not auto-reject without approval.
- HR must have the ability to review AI recommendations and explicitly approve rejection.
- Candidates do NOT log in.

- Interview scheduling should be manual inside the dashboard.
- Email invite functionality should exist.
- Outlook / Google Calendar integration should be designed for future extensibility, but not fully implemented now.

## Core Functional Modules

### 1. Job Management
- Create new job posting
- Edit existing job posting
- Archive / close job
- Fields should include:
  - title
  - department
  - location
  - employment type
  - seniority
  - required skills
  - preferred skills
  - years of experience
  - salary range optional
  - description
  - responsibilities
  - qualifications
  - status
  - created by
  - timestamps

### 2. Resume Upload and Candidate Profiles
- HR manually uploads resumes
- Support drag-and-drop upload
- Accepted formats: PDF, DOC, DOCX
- Parse extracted text from resume
- Generate candidate profile record automatically
- Candidate record should include:
  - full name
  - email if found
  - phone if found
  - skills extracted
  - experience summary
  - education summary
  - past roles
  - location if found
  - uploaded resume file path
  - parsed resume text
  - recruiter notes
  - linked job posting(s)

### 3. AI Resume Screening
For each uploaded candidate against a selected job description:
- Parse the resume
- Compare candidate profile to JD
- Generate match score
- Generate skill match breakdown
- Highlight missing skills / skill gaps
- Summarize candidate strengths
- Summarize risks / concerns
- Recommend one of:
  - Strong Fit
  - Moderate Fit
  - Weak Fit
- Never auto-reject silently
- HR must approve any rejection action

### 4. AI Candidate Ranking
For a job posting:
- Rank all associated candidates by fit
- Show score explanation
- Show matched skills
- Show missing skills
- Show concise reasoning
- Allow recruiters to sort and filter candidates

### 5. AI Job Description Support
Provide AI assistance to:
- Generate JD drafts from a short prompt
- Improve existing JDs
- Suggest required skills and qualifications
- Suggest interview focus areas

### 6. AI Interview Question Generator
Generate structured interview questions based on:
- the job description
- candidate resume
- candidate skill gaps
- experience level

Output categories:
- technical questions
- behavioral questions
- role-specific scenario questions
- follow-up probing questions

### 7. Candidate Summary
Generate AI summary for each candidate:
- overall profile summary
- experience snapshot
- strengths
- risks
- likely fit areas
- missing capabilities
- recommendation notes for recruiter

### 8. Interview Scheduling
- HR manually selects interview date/time inside dashboard
- Attach candidate to interview record
- Assign interviewer name
- Set interview type
- Set status
- Send email invite only
- Keep design ready for future Outlook / Google Calendar integration

### 9. Email Notifications
Implement email notification flows for:
- HR when candidate is uploaded
- HR when AI screening completes
- Candidate when interview is scheduled
- Candidate when application status is updated if recruiter chooses to send email
- Keep email templates modular and reusable
